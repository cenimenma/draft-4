## 💡 工作原理

```
开发者 git push
    ↓
GitHub Actions 自动触发（ai_review.yml）
    ↓
提取 git diff（ai_review.py）
    ↓
调用 DeepSeek API（deepseek-chat）
    ↓
生成审查报告
    ↓
通过 GitHub API 发布评论
    ↓
✅ Commit 页面出现 AI 审查意见
```

## 🐛 常见问题

**Q: Workflow 没有触发？**  
A: 检查 Actions 是否启用，是否推送到 main/master 分支

**Q: API 调用失败？**  
A: 确保 LLM_API_KEY secret 已正确设置，API Key 有效且有余额

**Q: 评论没有发布？**  
A: 检查 workflow 是否有 `pull-requests: write` 权限
